# Welcome to DocCrate

DocCrate is an **extremely fast** local documentation browser built on Direct2D and DirectWrite — no browser engine, no Electron, no web view overhead.

## Features

- Instant markdown rendering via Direct2D
- Click-to-navigate hyperlinks
- Keyboard navigation (arrows, Page Up/Down, Home/End)
- Sidebar file browser
- Back/Forward history (← →)
- Mouse-wheel and scroll-bar scrolling

## Navigation

Use the **sidebar** on the left to switch between pages.

Press **← Left Arrow** to go back, **→ Right Arrow** to go forward.

Mouse-wheel scrolls the content. The scroll bar on the right is draggable.
